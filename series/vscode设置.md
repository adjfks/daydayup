`.prettierrc`文件

```json
{
  "semi": false,
  "singleQuote": true,
  "arrowParens": "always",
  "trailingComma": "all"
}

```

设置分为`用户区`和`工作区`设置，用户区设置会应用在全部项目，工作区设置会生成`.vscode`文件，该设置只会在当前项目生效。配置完如果不生效可以重启一下vscode。



自动保存

设置搜索`save`将`auto save`设置为`onWindowChange`当窗口失去焦点时会自动保存。

搜索`默认格式化`可以设置默认格式化工具，比如设置`prettier`
